# Запуск проекта

```
docker compose up --build
```

# Миграции

Чтоб применить миграции откройте консоль контейнера
```
docker exec -it ntech-api bash
```

Перейдите в папку миграций и запустите их
```
cd db/migrations
alembic upgrade head
```
